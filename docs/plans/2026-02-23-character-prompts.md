## 치이카와 스타일 핵심 원칙

```
머리:몸 비율 = 1:1 ~ 1:1.5 (머리가 전체의 40~60%)
눈 = 얼굴의 1/3 이상, 반짝이는 하이라이트 필수
형태 = 원과 부드러운 곡선만 (각진 부분 절대 없음)
입 = 작고 단순하게 (ω형 또는 점)
색감 = 파스텔톤, 부드러운 색만
외곽선 = 두꺼운 외곽선 (2~3px), 짙은 회색(#4A4A4A)
```

## 프롬프트 작성 팁

### 일관성 확보
- 나노바나나: 첫 캐릭터가 마음에 들면 "이전 캐릭터와 같은 스타일로" 붙이기
- Midjourney: `--sref` (style reference) 파라미터로 첫 결과물 URL 참조

### 부정 프롬프트 (Negative)
```
사실적이지 않게, 그라데이션 없이, 3D가 아닌 2D,
복잡한 배경 없이, 비비드한 색 없이, 날카로운 모서리 없이
```

Midjourney:
```
--no realistic, 3D, gradient, complex background, vivid colors, sharp edges, photograph
```

### 수정 대화 예시 (나노바나나)
```
"눈을 20% 더 키워줘"
"볼터치를 더 진하게"
"몸을 좀 더 통통하게"
"같은 캐릭터를 옆모습으로 그려줘"
"같은 캐릭터가 앉아있는 포즈로"
"같은 캐릭터가 놀란 표정으로"
```

### SVG 변환 고려사항
- 단순한 형태, 적은 색상 수, 플랫 컬러 강조
- 최종 결과물은 vectorizer.ai 또는 Adobe Illustrator로 벡터 변환 가능
- 앱 내 사용 시 PNG(래스터) 또는 SVG(벡터) 선택
