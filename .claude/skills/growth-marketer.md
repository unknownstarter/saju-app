# Growth Marketer — 사주 데이팅 앱

## 1. 바이럴 루프 설계

### 핵심 바이럴 메커니즘

#### 1) 사주 카드 공유 루프 (Primary Viral Loop)
```
사용자 → 사주 분석 결과 확인 → "와 신기하다" → SNS 공유
→ 친구가 카드 봄 → "나도 해볼래" → 앱 설치 → 사주 분석
→ SNS 공유 → 반복

전환 포인트:
- 사주 결과 화면에 "공유하기" 버튼 최상단 배치
- 공유 카드 디자인: 인스타 스토리에 최적화된 세로 비율
- 카드에 QR코드 + 딥링크 삽입
- "나도 내 사주 보러 가기" CTA 텍스트

목표 K-factor: 공유율 15% × 클릭률 20% × 설치율 30% = 0.009
→ 100명당 0.9명 추가 유입 (초기, 이후 최적화로 증가)
```

#### 2) 궁합 초대 루프 (Secondary Viral Loop)
```
사용자 A → 특정 사람과 궁합 보고 싶음 → 초대 링크 전송
→ 상대방 B 앱 설치 → 생년월일시 입력 → 궁합 결과 표시
→ B도 다른 사람과 궁합 보고 싶어짐 → 초대 → 반복

전환 포인트:
- "이 사람과 궁합 확인하기" → 카카오톡/문자 초대
- 초대받은 사람은 가입만 하면 바로 궁합 결과를 볼 수 있음
- 궁합 결과에 "더 자세한 분석은 앱에서" 넛지

목표: 사용자당 평균 2.5명 초대 → 초대 수락률 40% = 1명 추가
```

#### 3) 데이팅 매칭 루프 (Retention Viral)
```
매칭 성사 → 대화 → 좋은 경험 → 친구에게 추천
→ "나 이 앱에서 사귀게 됐어" → 설치

이건 느린 루프이지만 가장 강력한 추천
```

### 바이럴 콘텐츠 아이디어
| 콘텐츠 | 형식 | 공유 동기 |
|--------|------|----------|
| 내 사주 분석 카드 | 이미지 | "나 이런 사람이래!" 자기 표현 |
| 오늘의 운세 | 이미지/텍스트 | "오늘 대박날래" 일상 공유 |
| 궁합 테스트 | 링크 | "우리 궁합 볼래?" 관계 확인 |
| MBTI × 사주 조합 | 이미지 | "ENFP인데 丙火래" 정체성 탐구 |
| 연애 스타일 분석 | 이미지 | "내가 연애하면 이래" 자기 공개 |
| 올해 연애운 | 이미지 | "올해 누구 만난다고?" 기대감 |

## 2. 유저 획득 채널 (한국 시장)

### 채널 우선순위

#### Tier 1: 핵심 채널 (집중 투자)

##### 인스타그램 (Instagram)
```
왜: 2030 여성 타겟, 비주얼 콘텐츠, 스토리 공유 문화
전략:
  - 릴스: 사주 분석 결과 리액션 영상 (15-30초)
  - 스토리: 오늘의 운세, 궁합 테스트 스티커
  - 피드: 사주 유형별 연애 패턴 인포그래픽
  - 광고: 사주 결과 샘플 + "내 사주 무료 분석" CTA

KPI: CPM ₩3,000-5,000, CPI ₩2,000-4,000
예산 배분: 전체 마케팅 예산의 35%
```

##### TikTok (틱톡)
```
왜: MZ 타겟, 숏폼 바이럴, 알고리즘 도달
전략:
  - 사주 분석 리액션 시리즈 ("연예인 사주 분석")
  - "궁합 테스트 결과가 이래요" 커플 콘텐츠
  - 사주 유형별 밈 ("목(木) 기운 강한 사람 특징")
  - 챌린지: #내사주분석 #사주궁합

KPI: CPM ₩1,500-3,000, CPI ₩1,500-3,000
예산 배분: 30%
```

##### 카카오톡 (KakaoTalk)
```
왜: 한국 국민 메신저, 궁합 초대 루프의 핵심 채널
전략:
  - 카카오 비즈보드 광고
  - 궁합 초대 링크 최적화 (카카오톡 공유 시 미리보기)
  - 카카오 채널 운영 (오늘의 운세 자동 발송)
  - 카카오톡 선물하기 (프리미엄 구독권)

KPI: CPC ₩200-500, CPI ₩3,000-5,000
예산 배분: 15%
```

#### Tier 2: 보조 채널 (테스트 후 확대)

##### 네이버 (Naver)
```
전략:
  - 네이버 블로그: 사주 관련 SEO 콘텐츠 (장기 전략)
  - 네이버 카페: 연애/사주 관련 카페 바이럴
  - 네이버 검색 광고: "사주 궁합", "무료 사주" 키워드
예산 배분: 10%
```

##### 유튜브 (YouTube)
```
전략:
  - 사주 전문가 콜라보 (1-2명)
  - 쇼츠: TikTok 콘텐츠 재활용
  - 프리롤 광고: "무료 사주 분석 앱"
예산 배분: 10%
```

#### Tier 3: 실험 채널
- 트위터(X): 사주 밈, 연애 관련 트윗
- 에브리타임: 대학생 타겟 (캠퍼스 마케팅)
- 블라인드: 직장인 타겟
- Reddit r/korea: 외국인 한국 관심층 (글로벌 확장 시)

## 3. ASO (App Store Optimization)

### 앱스토어 최적화

#### 앱 이름 & 부제
```
앱 이름: [브랜드명] - 사주로 찾는 나의 인연
부제 (iOS): AI 사주 궁합 데이팅
짧은 설명 (Android): 사주 궁합으로 찾는 운명의 인연 - AI 사주 분석 & 매칭
```

#### 키워드 전략
```
iOS 키워드 필드 (100자):
사주,궁합,데이팅,소개팅,운세,사주팔자,오행,연애,만남,매칭,AI,인연,운명,무료사주,사주앱

핵심 키워드 (검색량 × 경쟁도 분석):
1순위: 사주 궁합, 소개팅 앱, 무료 사주
2순위: 사주 앱, 운세 앱, 궁합 보기
3순위: AI 매칭, 데이팅 앱, 연애 앱
```

#### 스크린샷 전략
```
스크린샷 순서 (5장):
1. "사주로 찾는 운명의 인연" + 매칭 화면
2. "AI가 분석하는 나의 사주" + 사주 결과 화면
3. "궁합 87%! 서로를 빛나게 하는 관계" + 궁합 화면
4. "매일 새로운 인연을 추천해드려요" + 추천 카드
5. "오늘의 연애운은?" + 오늘의 운세

디자인 원칙:
- 첫 스크린샷이 가장 중요 (설치 전환의 50%)
- 텍스트 최소화, 핵심 가치만 전달
- 실제 앱 화면 + 설명 오버레이
- 밝고 따뜻한 톤 (신뢰감 + 설렘)
```

#### 앱 설명
```
첫 3줄이 가장 중요 (접히기 전에 보이는 부분):
"사주로 찾는 운명의 인연!
AI가 분석한 사주 궁합으로 나와 가장 잘 맞는 사람을 만나보세요.
생년월일시만 입력하면 무료로 내 사주 분석 결과를 확인할 수 있어요."

이후: 주요 기능 나열, 후기/수상, 구독 정보
```

#### 리뷰 관리
```
리뷰 요청 타이밍:
- 매칭 성사 후 (긍정적 감정 피크)
- 사주 분석 결과 확인 직후 ("신기하다!" 순간)
- 3번째 접속 시 (초기 이탈 아닌 활성 사용자)

부정 리뷰 대응:
- 24시간 이내 응답
- 문제 인정 + 해결 약속
- 업데이트 후 재검토 요청
```

## 4. 레퍼럴 프로그램

### 프로그램 구조
```
초대하는 사람 (Referrer):
  - 친구 1명 가입 시: 프리미엄 3일 무료 체험
  - 친구 3명 가입 시: 프리미엄 7일 무료
  - 친구 5명 가입 시: 프리미엄 1개월 무료
  - 친구 10명 가입 시: VIP 1개월 무료

초대받는 사람 (Referee):
  - 가입 시: 프리미엄 3일 무료 체험
  - 프로필 완성 시: 슈퍼좋아요 3개 지급
```

### 레퍼럴 메커니즘
```
1. 고유 초대 코드 생성 (예: SAJU_NOAH_7X9K)
2. 초대 링크 공유 (딥링크: sajuapp://invite?code=SAJU_NOAH_7X9K)
3. 공유 채널:
   - 카카오톡 (기본)
   - 문자 메시지
   - 인스타그램 DM
   - 링크 복사
4. 추적: 초대 코드 → 설치 → 가입 → 프로필 완성 → 보상 지급
```

### 궁합 기반 특수 레퍼럴
```
"이 사람과 궁합 확인하기" 플로우:
1. A가 특정 사람(B)을 선택해 궁합 초대 전송
2. B가 링크 클릭 → 앱 설치 → 생년월일시 입력
3. A와 B 모두에게 궁합 결과 표시
4. B에게 "앱에서 더 많은 인연을 찾아보세요" 넛지

이 루프가 일반 레퍼럴보다 전환율 2-3배 높을 것으로 예상
(구체적인 동기 + 즉각적인 가치 제공)
```

## 5. 리텐션 훅 (Retention Hooks)

### Daily Hooks (매일 돌아오게 하는 것)

#### 오늘의 운세
```
매일 아침 8시 푸시 알림:
"오늘의 연애운을 확인해보세요 ✨"
→ 앱 진입 → 운세 확인 → 자연스럽게 매칭 탐색

콘텐츠 변화:
- 매일 다른 운세 (일진 기반)
- 행운 색상, 행운 숫자, 행운 방향
- 연애 팁 (사주 기반)
- "오늘 이런 사람을 만나면 좋아요" 매칭 연계
```

#### 새로운 매칭 알림
```
"새로운 인연이 도착했어요! 사주 궁합 92%"
→ 호기심 → 앱 진입 → 프로필 확인

전략:
- 매일 정해진 시간에 새 추천 (점심 12시, 저녁 8시)
- 궁합 점수를 미리 보여줘서 기대감 유발
- "오늘의 베스트 매칭" 하이라이트
```

#### 대화 알림
```
"[닉네임]님이 메시지를 보냈어요"
→ 대화 지속 → 관계 형성 → 앱 가치 체감

전략:
- 읽지 않은 메시지 뱃지
- 24시간 미응답 시 리마인더
- "사주 기반 대화 주제" 추천으로 대화 활성화
```

### Weekly Hooks (주간)
```
- 주간 사주 운세 리포트 (매주 월요일)
- "이번 주 당신에게 관심을 보인 N명" (프리미엄 넛지)
- 주간 인기 프로필 (동기부여)
```

### Milestone Hooks (달성 기반)
```
- 첫 번째 매칭 축하 🎉
- 10번째 좋아요 달성
- 대화 100회 달성
- 1개월 사용 기념 사주 리포트
```

## 6. 퍼널 최적화

### 전체 퍼널
```
광고 노출 → 클릭 → 스토어 → 설치 → 첫실행 → 가입 → 사주입력
→ 프로필완성 → 첫매칭탐색 → 첫좋아요 → 매칭성사 → 첫메시지
→ 5회대화 → 7일리텐션 → 구독전환
```

### 각 단계별 최적화

#### 설치 → 가입 (목표: 60%)
```
장벽: "또 회원가입이야?" 피로감
해결:
  - 카카오 원터치 로그인 (가장 익숙한 방법)
  - 약관 동의 간소화 (전체 동의 1탭)
  - 가입 전 사주 분석 맛보기? (AB 테스트 필요)
  - 가입 화면에 "30초면 끝나요" 메시지

측정:
  - 가입 시작률
  - 로그인 방법별 전환율
  - 약관 동의 단계 이탈률
```

#### 가입 → 사주 입력 (목표: 85%)
```
장벽: "생시를 모르겠어요"
해결:
  - "모름" 옵션 명확히 제공
  - "부모님께 여쭤보세요" 카카오톡 질문 템플릿
  - 생시 없이도 의미 있는 분석 제공
  - 양력/음력 자동 판별 도움

측정:
  - 생시 입력률 vs 모름 선택률
  - 양력/음력 비율
  - 입력 완료 소요 시간
```

#### 사주 입력 → 프로필 완성 (목표: 65%)
```
장벽: "사진 올리기 귀찮아", "자기소개 뭐라 쓰지"
해결:
  - 사주 기반 자동 프로필 초안 생성 ("목(木) 기운의 성장형 리더")
  - 사진 1장만 필수 (나머지는 나중에)
  - 자기소개 대신 Q&A 형식 ("주말에 주로 뭐 해요?")
  - 프로필 완성도 게이지 (심리적 완성 욕구 활용)

측정:
  - 사진 업로드률
  - 평균 사진 수
  - 자기소개 작성률
  - 프로필 완성 소요 시간
```

#### 첫 매칭 탐색 → 첫 좋아요 (목표: 80%)
```
장벽: "맘에 드는 사람이 없어", "좋아요 누르기 부담"
해결:
  - 첫 추천 품질 최대화 (궁합 높은 사람 우선)
  - 궁합 점수로 관심 유발
  - "궁합이 좋은 상대에요!" 배지 강조
  - 좋아요 부담 감소: "상대에게는 익명이에요" 안내

측정:
  - 첫 카드 스와이프까지 시간
  - 좋아요/패스 비율
  - 첫 세션 탐색 카드 수
```

#### 매칭 성사 → 첫 메시지 (목표: 60%)
```
장벽: "뭐라고 말하지", "먼저 연락하기 부담"
해결:
  - 사주 기반 아이스브레이커 3개 자동 추천
    - "두 분 다 火 기운이 강해서 열정적인 대화가 기대돼요! 최근에 가장 열정 느낀 순간이 있어요?"
  - 매칭 후 24시간 타이머 (긴박감)
  - 매칭 성사 애니메이션 후 바로 채팅방 이동
  - "첫 메시지를 보내면 궁합 상세 분석을 볼 수 있어요" 인센티브
```

## 10. Cold Start 문제 해결

### 데이팅 앱의 근본 문제
```
사용자가 없으면 매칭이 안 됨 → 매칭이 안 되면 사용자가 떠남
= 닭과 달걀 문제
```

### 해결 전략

#### 1) 사주 분석 = 독립적 가치
```
핵심: 매칭 없이도 앱에 가치가 있어야 함

사주 분석 결과만으로도:
- "내 성격 유형이 이래?"
- "올해 연애운이 이래?"
- "SNS에 공유해야지"

→ 매칭 풀이 작아도 이탈하지 않음
→ 시간을 벌어 사용자 수를 키울 수 있음
```

#### 2) 지역 집중 전략
```
서울 → 수도권 → 광역시 순서로 확대
한 지역에서 충분한 밀도 확보 후 다음 지역으로

밀도 기준: 같은 지역 내 활성 사용자 500명 이상
```

#### 3) 성비 관리
```
데이팅 앱 성비 관리 전략:
1. 여성 사용자 먼저 확보 (사주 분석/운세 콘텐츠로)
2. 여성:남성 = 6:4 이상 유지 시 남성 유입 허용
3. 남성 과잉 시 남성 신규 가입 웨이트리스트

인센티브:
- 여성: 프리미엄 기능 더 많이 무료 제공
- 남성: 프로필 완성도/인증 요구 높이기 (품질 필터)
```

#### 4) 궁합 바이럴 = 유저 유입 엔진
```
기존 사용자가 앱 밖의 사람에게 궁합 초대 → 신규 유입
이 루프가 돌면 자연스럽게 유저 풀 확대
```

## 13. K-Factor & 핵심 성장 공식

### K-Factor 계산
```
K = i × c
  i = 사용자당 평균 초대 수
  c = 초대 → 가입 전환율

목표:
  Phase 1 (론칭): K = 0.3 (바이럴 보조적)
  Phase 2 (3개월): K = 0.6 (바이럴이 성장 엔진)
  Phase 3 (6개월): K > 1.0 (자연 성장)
```

### K-Factor 개선 레버
```
i (초대 수) 개선:
  - 사주 카드 공유 CTA 최적화
  - 궁합 초대 프롬프트 추가
  - 레퍼럴 보상 강화
  - 공유하기 좋은 콘텐츠 지속 생산

c (전환율) 개선:
  - 공유 카드 디자인 최적화 (클릭 유도)
  - 딥링크 → 앱스토어 → 앱 진입 플로우 매끄럽게
  - 초대받은 사람의 첫 경험 최적화 (즉시 가치 전달)
  - 앱스토어 전환율 최적화 (ASO)
```

### 성장 모델
```
월 신규 사용자 = 유료 획득 + 오가닉 + 바이럴
  = (광고 예산 / CPI) + (ASO × 검색량) + (기존 유저 × K-factor)

예시 (론칭 후 3개월):
  유료: ₩10M / ₩3,000 = 3,333명
  오가닉: 1,000명
  바이럴: 10,000 × 0.5 = 5,000명
  ─────────────────
  총: 9,333명/월

목표: 론칭 6개월 내 MAU 50,000명
```
